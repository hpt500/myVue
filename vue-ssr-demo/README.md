# vue-ssr-demo

> A Vue.js project

>> 关于vue项目构建的初始模板其中包括

>>> ssr客户端及其服务器端的配置

>>> dev模式与ssr的分开配置

>>> sass的配置

## Build Setup

### Start

``` bash
# install dependencies
npm install
```

### dev 模式
``` bash
# 在localhost:8080运行dev服务
npm run dev

# 打包项目
npm run build

# 打包报告
npm run build --report
```

### ssr 模式
``` bash
# 打包renderer(包括客户端client及服务器端server)
npm run build:all

# 通过node express运行服务器端
node server.js
```
